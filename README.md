Projeto de cadastro de cupcakes


## Como iniciar

-**(GIT clone https://github.com/PauloSPires/PITII)**
-**(cd PITII)**
-**(copy .env.exemple .env)**
-**(composer install)**
-**(npm install)**
-**(npm run prod)**
-**(copy null > database.sqlite)**
-**(php artisan migrate)**
